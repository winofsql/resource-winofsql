```sql
GRANT ALL PRIVILEGES ON *.* 
	TO root@'%'

```

## パスワード設定
```sql
GRANT ALL PRIVILEGES ON *.* 
	TO ユーザ@'Host'
	IDENTIFIED BY 'password'
	WITH GRANT OPTION

```
**※ 登録されていないユーザは新規登録**

## ユーザ削除
```sql
drop user ユーザ@'Host'
```

    `pdate` datetime,

    PRIMARY KEY(`row_no`)
